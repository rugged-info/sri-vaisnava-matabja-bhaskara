# Śrī Vaiṣṇava Matābja Bhāskara  
### Sanskrit–English Translation Project

Welcome. This site presents my ongoing Sanskrit–English translation of the *Śrī Vaiṣṇava Matābja Bhāskara*, using a consistent translation protocol on each verse:

1. Sanskrit (Devanāgarī)  
2. IAST transliteration  
3. Word-by-word breakdown with grammar  
4. Literal translation  
5. Smooth English translation  
6. Optional commentary and notes
